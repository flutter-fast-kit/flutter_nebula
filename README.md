# flutter_nebula

Flutter nebula 是 Eva 设计系统的一个Flutter实现。它包含一组以类似方式设计的通用 UI 组件。您专注于业务逻辑，Nebula 负责视觉外观。最棒的是: 主题可以在运行时更改，而不需要重新加载应用程序。

## 开始使用

```yaml
flutter_nebula:
    git: 
      url: git@gitlab.tqxd.com:aitd_exchange/aitd_exchange_mobile/flutter/flutter_nebula.git
      ref: v1.0.0
```
